포커스 들어오는 순간 토스트 사라지고, 키보드가 결과 목록만 squeeze.
규칙 요약
통일 규칙 — 인풋은 항상 하단 출발/도착 바(od-bar) 하나만. 상단 검색바 없음.
A (기본 구조) — flex column: 결과 목록(flex:1) + od-bar + 키보드. 키보드가 올라오면 목록이 squeeze됨.
B (하지 말 것) — 인풋을 absolute로 지도 위에 floating. 키보드가 그냥 덮어버림.
C (토스트) — 인풋 포커스 시 임시 알림 즉시 dismiss.
